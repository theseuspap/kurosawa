# Instructions for R script execution

1st part of code: loading libraries
Please set your working directory in line 23

2nd part of code: setting up connection to Google BigQuery and fetching data

3rd part of code: data manipulation and commands are done through the data.table library

a. k-means part can be executed and summary of results can be found within the script
b. k-proto starts after line 175 and the results are exported in the end of the scripts. These two files are the ones used for Power BI dashboard
